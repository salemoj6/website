The CPU is the brains of the computer, without it nothing would work. When putting together a computer, depending on what you want the CPU is a very important component to consider, since it could limit you from using other parts you might want. Not only that, but different CPUs have different strength, such as for gaming you might want a more powerful single core CPU (can put more effort into one task) or if you want to do many things at once, like having multiple applications running or even working on a creative project like video editing, recording, streaming, etc., it could be more important to have a multi-core processor.


When you think of storage with computers, usually one thinks about KBs, MBs, and even GBs to store files and the sort, but RAM is a different type of storage. Since Hard Drives and Solid State Drives aren't fast enough for certain computer processes, we need something faster, and that's where RAM comes in. RAM does also store data, but unlike other storage types, it is much faster but can't keep data after the power is turned off. That's why it's used for processes where it needs to store, access, and then throw away data quickly. It can also send data to permanent storage if told, and certain games like Minecraft use a lot of RAM to work.


Operating System choice is mostly up to preference, but it can also be a matter of how easy it is to get applications running on it. Most of the time I recommend Windows10 since it's easy to use and most gaming/creative applications work natively with it. If you have computer skills you may also opt into running Linux in one of the many custom made Operating Systems on the web, but it will be more tedious to try and get a program called Wine to run Windows apps.


A Graphics Card is optional to those who just want to browse the web, and are quite expensive nowadays, but if you plan on getting the best AMD cpu or want to game on the PC, a GPU is highly recommended, if not mandatory for some.


A Motherboard is used as a way for all the components in the PC to communicate with one-another, and for your input devices to be plugged in to use through USB, Ethernet, and other connectors. The only really important things to know are to make sure your CPU picked works with it, it has enough slots to accommadate the other parts, and it can handle certain speeds. Also making sure if you want Bluetooth/WIFI capabilities.



Most people nowadays know about files and storing them over a long period of time, but there's also the speed of said storage device that's important. Having a faster storage device like a Solid State Drive(SSD) can be important if you want to load files faster, but is very expensive and cannot store as much as a Hard Disk Drive(HDD) which is usually cheaper for large amounts of data storage, but isn't as fast and can be quite loud when the disks spin inside, and can become damaged through vibrations because it has moving parts inside.


All you need to know is that a Power Supply(PSU) is just used to bring enough electricity to each component. NEVER cheap out on buying a PSU since it could explode if something goes wrong, so always make sure it has a bronze, silver, or gold certification on it, and that it has enough wattage for all the components combined. PCPartPicker is really good for figuring those numbers out
